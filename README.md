# Strudel Algorave Workspace

Algorithmic music compositions using [Strudel](https://strudel.cc/), a browser-based live coding environment for music (JavaScript port of TidalCycles).

## How to Play

1. **Browser:** Copy the contents of any `.strudel` file and paste into [strudel.cc](https://strudel.cc/)
2. **Neovim:** Open a `.strudel` file in Neovim with strudel.nvim installed, press `<Space>sl` to launch

## Setup

Requires [strudel.nvim](https://github.com/gruvw/strudel.nvim) for the Neovim workflow. See `CLAUDE.md` for full syntax reference and keybindings.
